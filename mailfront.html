MailFront

SMTP Front Ends
POP3 Front End

Overview

This package contains customizeable network front-ends for mail servers. Right now the only protocols that are handled are SMTP and POP3, but this package is likely to be extended to deal with QMQP and QMTP, and potentially even IMAP.

Status

This package is ready to use, but you will have to make some modifications to existing configurations to do so. smtpfront-qmail is a nearly complete drop-in replacement for qmail-smtpd, and pop3front-auth and pop3front-maildir are replacements for qmail-popup and qmail-pop3d.
